1. VS Code or Visual Studio [Download Visual Studio Code](https://code.visualstudio.com/download)
2. .NET SDK (latest stable version) [Download and install .NET SDK (latest stable version)](https://dotnet.microsoft.com/en-us/download/dotnet/8.0)
3. GIT [Git - install](https://git-scm.com/install/)
4. Setting your username and email in Git [Git - setup](https://learn.microsoft.com/en-us/devops/develop/git/install-and-set-up-git#configure-git-on-linux)
5. Setup UserName and Password. Do not modify `appsettings.json` directly. Use `appsettings.local.json` for local overrides.
